#### Git commands w/short descriptions:

1. git init - creates an empty git repository, or reinitializes an existing one
2. git status - displays the paths of the Git tree
3. git add - add file contents to the index
4. git commit - records changes to the repository
5. git push - uploads local repository content to a remote repository, such as Bitbucket
6. git pull - uploads remote repository content to local repository (vice versa of push)
7. git clone - creates a new, local Git repository in the current directory
