git init - creates an empty git repository, or reinitializes an existing one
git status - displays the paths of the Git tree
git add - add file contents to the index
git commit - records changes to the repository
git push - uploads local repository content to a remote repository, such as Bitbucket
git pull - uploads remote repository content to local repository (vice versa of push)
git clone - creates a new, local Git repository in the current directory
